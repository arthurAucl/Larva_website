Installation required


AspectJ

To use the compiler, you need to install AspectJ and run your code with AspectJ. The version of AspectJ depends of your version of Java.
You can find all version on this site : AspectJ download .

On eclipse, go to "Help", "Install new sotfware" and click on "add" and put the link of your version.



How to use the compiler

To use the compiler, you need to install AspectJ and run your code with AspectJ.
After, go to "run configuration" :


Then, you compile with AspectJ and you put your project (Larva) and the main class (compiler) :


Now, you have to put some arguments. Go to arguments, write the directory of the larva file you want to execute, write -v -o and to finish write the directory of your src file.
Then, run the compilator.


Normally, you need to have "Compiled successfully" in the console.

Graphviz problem

It's possible that you have an error with Graphviz when you compile.
If you have this error, you can put the following part of code in the property.java file at the line 140 (directory : Larva/src/compiler/property.java) : 

                                try{
                                    PrintWriter pw1 = new PrintWriter(Compiler.outputDir+"diag"+name+".txt");
                                    pw1.write(sb.toString());
                                    pw1.close();
                                
                                    String command = Compiler.graphvizDir
                                        + " -Tgif"
                                        + " -o " + outputDir + "_logic" + name + "_diag.gif"
                                        + " -Kdot " + Compiler.outputDir + "diag"+ name + ".txt";
                                    Runtime.getRuntime().exec(command);
                                    
                                    }

Example

In this part, we will show you an example with a bank system. The goal of this example is to create users, account and transactions but also to make sure that you can't have more than 6 user, each user can't have more than 6 account and each account can't have more than 6 transactions.

When you will run the bank example, this is what you will have in the console of eclipse :


Here you can choose what you want to do. For example you can type 1 to add user or 3 to delete one user.
We can verify that the user are correctly created in the file "output_bank.txt" :


In this example, it's impossible to create more than 6 users. Let's try to create 6 users :


Here, after that we create 7 users, we can see an error in the output file : "SYSTEM REACHED BAD STATE"
and it's written that we created too many users.
