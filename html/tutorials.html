Installation required

AspectJ

To use the compiler, you need to install AspectJ and run your code with AspectJ. The version of AspectJ depends of your version of Java.
You can find all version on this site : AspectJ download .

On eclipse, go to "Help", "Install new sotfware" and click on "add" and put the link of your version.
